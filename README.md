# baking-converter
A web application that converts baking measurements from cups to grams. Since The U.S wants to be different :)

>**Single Ingredient Converter** - Convert any ingredient from cups to grams"
>
>**Recipe Scaler** - Scale entire recipes by a custom multiplier (0.25x to 4x)
>
>**Batch Converter** - Convert multiple ingredients at once
>
>**Conversion History** - Automatically logs all conversions with timestamps
>
>**Export Options** - Download history as CSV or JSON
>
>**Reference Table** - Quick lookup for conversion rates
>
